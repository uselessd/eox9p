Accessors for 9p connection parameters.

Description

Accessors for 9p connection parameters. All functions of a callback module that implements an exported directory take and opaque Conn parameter. The '9p_info':* functions provide access to various bits of information stored in the Conn structure. The primary purpose is to make informed decisions about permissions of files and directories.
